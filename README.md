# 贪吃蛇小游戏

一个使用HTML5 Canvas开发的贪吃蛇游戏，专为移动设备优化，可通过GitHub Pages部署。

## 游戏功能

1. **主菜单页面**
   - 游戏标题"贪吃蛇"
   - "开始游戏"按钮
   - "最高纪录"按钮
   - 游戏说明

2. **游戏页面**
   - 游戏画布显示蛇、苹果和游戏元素
   - 分数和苹果计数显示
   - 游戏时间计时器
   - 移动设备友好的方向控制按钮
   - 游戏结束弹窗显示本次分数和"返回菜单"按钮

3. **游戏机制**
   - 控制蛇吃苹果增长长度和分数
   - 每个小苹果增加1分
   - 每吃5个苹果出现一个限时大苹果（10分）
   - 蛇头碰到墙壁或自己的身体游戏结束

4. **最高纪录**
   - 显示前5名最高分数
   - 使用浏览器本地存储保存记录
   - 包含分数、苹果数量和日期信息

## 如何部署到GitHub

1. 在GitHub上创建一个新仓库
2. 将所有文件上传到仓库
3. 进入仓库设置 > Pages
4. 选择部署分支（通常是main/master）
5. 保存后等待部署完成
6. 访问提供的URL即可游戏

## 游戏控制

- 桌面：使用键盘方向键（↑↓←→）
- 移动设备：点击屏幕上的方向按钮
- 空格键：开始游戏（桌面）

## 技术栈

- HTML5 Canvas
- CSS3（Flexbox，Grid，响应式设计）
- JavaScript（ES6+）
- 本地存储API

## 浏览器支持

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）

## 自定义

- 修改`style.css`调整游戏外观
- 修改`script.js`中的`gameSpeed`调整游戏速度
- 修改`script.js`中的`tileCount`调整游戏网格大小
